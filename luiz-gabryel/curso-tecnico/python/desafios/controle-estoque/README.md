# 📚 Sistema de Controle de Estoque de Livros

Projeto em Python para gerenciamento de livros via terminal, com sistema de login e menu interativo.

## Sobre o projeto

Este sistema foi criado como prática de lógica de programação e aprofundamento no uso da linguagem Python.
Ele permite cadastrar, visualizar, buscar e remover livros dentro de um controle de estoque simples.

O projeto ainda está em desenvolvimento e receberá melhorias com o tempo.

## Funcionalidades

- Login com usuário e senha
- Validação de campos vazios no login
- Visualização de todos os livros cadastrados com quantidade
- Adição de novos livros ao estoque com quantidade
- Busca de livro por nome
- Remoção de livro com confirmação de senha
- Deslogar com confirmação de senha
- Aviso para opções inválidas no menu

## Como executar

1. Abra o terminal
2. Navegue até a pasta do projeto
3. Execute:

```bash
python controle_estoque.py
```

## Estrutura do sistema

O programa funciona em modo terminal e utiliza:

- Lista em memória para armazenar livros e quantidades
- Loop principal para manter o sistema ativo
- Verificação de login antes de acessar o menu
